Using the REST API¶

This document describes the structure of the REST API.

atencoding¶

Since some characters have special meanings in URLs it is necessary to have an encoding that will allow these characters to pass through unchanged. This encoding is analgous to the `percent encoding`_ as used in URIs and defined in RFC 3986.
